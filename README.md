# WooCommerce Product Reviews

Enhance your WooCommerce store with **better product reviews**. This plugin adds:
- **Custom review title** field
- Clean and professional **review display**
- Fully **translation-ready** (.pot file included)
- Compatible with latest **WooCommerce versions**

## Features
- Easy to use for beginners
- Boosts credibility with detailed reviews
- SEO-friendly product reviews for better search visibility
